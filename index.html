Web Bluetooth Demo

This webapp demonstrates the Web bluetooth Follow these steps to use the demo:

  1. :
  2. Trigger a request:
    1. Check the verbose box if you want debug output.
    2. The default advertising filter is 'TYSC-'. Enter a full BLE friendly name if you want to restrict the filtering to a single BLE Device.
    3. Click on the Execute button.
      A BLE connection will be established to the BLE peripheral device and the response will be show in the verbose window.
      The BLE connection will be dropped after the response is received.
  3. Repeat.
